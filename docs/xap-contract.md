# XAP/XIP Script Contract

This document catalogs every node type, function, property, and callback that XAP scripts in the dashboard XIPs reference. This is the "do not break" API surface; the C++ implementation behind these interfaces can be rewritten freely, but the names and behaviors must remain stable.

Generated by scanning all 39 .xap files across the dashboard XIP set.
